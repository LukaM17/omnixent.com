## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ export NUXT_ENV_API_SERVER=https://youromnixentserver/api/v1/
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

## Run tests
```
$ npm run ci
```
